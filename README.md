# アプリ名
## SAMPLE_BULLETINBOARD_APP

## 実装機能
  * ログイン機能 (email + pass)
  * 閲覧・投稿機能
  * スレッド機能 (スレッドにレスをつけていくスタイル)
  * カテゴリ(タグ)設定 (スレッドごと / 1つのスレッドに複数付与可能)
  * 検索機能 (単語 / スレッド及びレスの横断検索)
